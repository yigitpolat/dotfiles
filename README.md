# dotfiles

## 1. Install Xcode developer tools

```bash
xcode-select --install
```

## 2. Clone repository

```bash
git clone https://github.com/yigitpolat/dotfiles.git ~/Documents/GitHub/dotfiles
cd  ~/Documents/GitHub/dotfiles
```

## 3. Brew install

```bash
make brew
```

## 4. Sync 

```bash
make sync
```
